Arquitectura

Exterior (HTTPS) dist0 (Nginx HTTPS/HTTP) <-> dist1/dist2 (Nginx HTTP) <-> (Flask HTTP)dist1/dist2 <-> sdist3 (NFS).

  • Resiliencia: dist1/dist2
  • SSL/TLS: Activo (Puerto 443)
  • Balanceo: Round Robin 3:1 (dist1/dist2)
  • Proxy Inverso: Activo (Puerto 80)
  • Servidor Web: dist1/dist2
  • Microservicios: 3 (Editoriales, Autores, Libros)
  • Almacenamiento: 1 (NFS scope database)

Repositorio

Estructura

Rama NFS: contiene la web estática y los datos.

Rama services: contiene los servicios dockerizados.

Rama gateway: contiene la configuración de Nginx y un generador de certificados.

Rama main: contiene un esqueleto como muestra.
